3、Monkey for GitHub
Monkey主要是用来展示GitHub上的开发者的排名，以及仓库的排名，并且能够搜索开发者和仓库。
开发者的排名是根据开发者的followers由高到低进行排名，有区分不同的语言和城市。
仓库的排名是根据仓库的stars由高到低进行排名的，有区分不同的语言。
这些数据是从GitHub提供的接口来的，地址是https://developer.github.com/v3/
